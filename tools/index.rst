================================
IronRuby tools for Visual Studio
================================
`IronRuby tools`_ provide a
*Ruby code editor*, a 
*file-based project system*, and an 
*interactive Ruby interpreter*, 
all inside `Visual Studio 2010 <http://www.microsoft.com/visualstudio/>`_, and 
**all for free**.
